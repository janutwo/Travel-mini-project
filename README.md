# MBTI-Travel

# 스프링 스터디 및 MBTI 테마 여행 모임 웹페이지

## 기획 의도
매번 새로운 친구와 떠나는 여행을 기획하는 친구의 일정과 파일,다녀온 후에 사진들을 카톡에서 관리하는 것의 불편함 해결
참여자들의 MBTI 성향을 파악하고 더 많은 사람들이 더 쉽게 참여하도록 하는 것 이 목표
## 요구 기능

1. 여행 일정 등록 
2. 여행 관련 문서 등록(.pdf,.xlsx)
3. 여행 후 단체 사진 게시 
4. 로그인
5. 여행 리뷰 커뮤니티
6. 참여자 중 MBTI 성향 적합도 표시 기능
